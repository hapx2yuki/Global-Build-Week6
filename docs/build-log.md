## 2026-07-21 GPT‑5.6実測

- 最初の実行群では、ある実行が「根拠へ到達できない」と解釈し不安定になった。送信前に端末内で検証済みの抜粋であり、返却後にも再検証する境界を指示へ明記した。
- 次の実行群では品質項目の`mustPass`表現が`pass`と`not_applicable`に揺れた。品質項目は品質段階だけで表し、必須判定を`not_applicable`へ決定論的に正規化した。
- 最終実行群は構造修復0回で`stable / not_met`に集約した。出力ハッシュは3回とも異なるため、完全決定性を主張していない。

## 2026-07-21 画面検証で見つけて直した点

- 分割領域の比率を数値で渡したため画素として解釈され、質問パネルが画面外へ押し出された。比率を文字列で指定して解消した。
- `CommandDialog` 内に `Command` の文脈がなく、`⌘K` で入力部品が失敗した。公式の構成に合わせて明示的に組み合わせた。
- スマートフォンでタブと質問ボタンの表示文字が隠れ、読み上げ名が空になった。幅に依存しない `aria-label` を追加した。
- スマートフォンの作成可能性5条件が縦に並び、本文を覆いすぎた。小画面では名称を読み上げ名に残し、印を横一列にした。
- 最終確認では、390×844の主要経路と側面ナビゲーションを操作でき、ブラウザのエラー0件・警告0件だった。
